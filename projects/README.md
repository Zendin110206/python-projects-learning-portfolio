# Projects

Project folders are added here one at a time as each exercise starts.

Expected folder convention:

```text
projects/
└── 01_project_name/
    ├── README.md
    ├── src/
    └── tests/
```

The exact structure may change for projects that need assets, data, or a different runtime layout.
